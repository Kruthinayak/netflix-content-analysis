# 🎬 Netflix Content Analysis Dashboard

## 📊 Project Overview

This project analyzes Netflix content data using SQL and Power BI to uncover meaningful insights about content distribution, growth trends, and audience ratings.

The dashboard is designed with a Netflix-inspired dark theme and provides interactive filtering for better data exploration.

---

## 🛠 Tools & Technologies

* SQL (Data Cleaning & Analysis)
* Power BI (Data Visualization)
* CSV Dataset

---

## 📈 Key Insights

* Movies dominate Netflix compared to TV Shows
* Significant growth in content after 2015
* United States and India contribute the most content
* Drama and International genres are highly popular
* Majority of content falls under TV-MA rating

---

## ✨ Features

* Interactive dashboard with slicers (Country, Type, Release Year)
* KPI cards (Total Titles, Movies, TV Shows)
* Clean Netflix-themed UI (Black + Red)
* Dynamic filtering across visuals
